# 智能招聘匹配平台

使用 Next.js、TypeScript、App Router 和 Tailwind CSS 构建的招聘匹配平台。

## 本地运行

安装依赖：

```bash
npm install
```

启动开发环境：

```bash
npm run dev
```

打开浏览器访问：

```text
http://localhost:3000
```

## 主要文件

- `app/page.tsx`：首页
- `app/layout.tsx`：页面布局和元信息
- `app/globals.css`：全局样式和 Tailwind CSS 引入
- `tailwind.config.ts`：Tailwind CSS 配置
- `postcss.config.mjs`：PostCSS 配置

## 常用命令

```bash
npm run dev
npm run build
npm run lint
```
